Mail.js

What

Mail.js is an open-source JavaScript library designed to help format email.

Note that Mail.js is not a mail transport -- Mail.js works with mail transport packages like Nodemailer.

Why

Email is notoriously hard to format for correctly. Here are some examples of the problems:
  • Most web mail readers support body <style> except gmail
  • Formatting both text and HTML email messages without message duplication can be painful

How

Mail.js uses a simple template system which allows you to simply craft your emails while making it easy to generate both text and HTML messages from the same source. It also bakes in HTML Email Boilerplate behind the scenes to solve many of the email rendering idiosyncrasies across email renderers.

Install

npm install mailjs --save

Mail.js is hosted on Github.
